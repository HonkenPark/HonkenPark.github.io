코드 한 줄보다 '무엇을, 어떤 구조로 만들 것인가'에 더 집중합니다. AI는 속도를, 설계와 검증은 사람이 담당한다고 믿습니다.

  1. 2011 — 2018

    인포테인먼트 & 미들웨어

    현대자동차 고급형 AVN에서 Qt/QML 미들웨어 개발과 SW 구조 설계, PL 업무를 수행했습니다. 해외 품질 평가 이벤트를 전담하며 글로벌 협업 경험을 쌓았습니다.

  2. 2018 — 2021

    글로벌 음성 시스템

    폭스바겐 프로젝트에서 Cerence 음성 엔진 포팅과 Amazon Alexa 연동 SAC를 담당했습니다. 독일·미국 개발팀과 주간 협업하며 실시간 오디오 처리와 C++11 기반 안정성 개선을 이끌었습니다.

  3. 2021 — 2023

    웹·키오스크 전환

    제주도청·성동구 민원 키오스크로 웹 영역을 확장했습니다. 정부 과제에 React를 최초 도입하고, Figma 기반 UI 협업으로 일정을 2주 단축했습니다.

  4. 2023 — 2025

    관제·제품 리딩

    삼성 DS 무인보관함 관제에서 JSP 레거시를 Flutter(MVVM)로 전환하고 화면·모듈 구조를 재설계했습니다. 일정 30% 단축, WebSocket 스트리밍으로 트래픽 40% 절감. ThinkAI 번역 솔루션·사내 자동화 툴 등 제품 프로젝트를 리드하며 관제·키오스크·대시보드를 end-to-end로 연결했습니다.

  5. 2025 — 현재

    AI 연계 설계 & IoT 관제

    제주패스·오에전기 KeepUs Go B2B 관제에서 Flutter·GetX MVVM으로 테넌트용 웹/모바일 관제를 설계·구현했습니다. ThingsBoard·커스텀 API 연동으로 차트 API 호출을 약 95% 줄였고, AI 도구를 개발 워크플로에 통합해 구조 설계와 산출물 검증을 맡고 있습니다.
